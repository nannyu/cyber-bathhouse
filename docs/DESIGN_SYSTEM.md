# 🎨 设计系统文档

赛博澡堂的视觉设计系统：像素风 + 赛博朋克美学。

---

## 1. 设计原则

1. **像素至上** — 所有图形元素遵循像素对齐，禁止抗锯齿
2. **霓虹赛博** — 暗色背景 + 高饱和度霓虹色彩
3. **蒸汽氛围** — 蒸汽粒子、水波纹增加场景生动感
4. **信息清晰** — UI 文字可读性优先于装饰性

## 3. 字体

| 用途 | 字体 | 降级 |
|------|------|------|
| 像素文字（标题/气泡） | `'Press Start 2P'` | `monospace` |
| UI 文字（聊天/表单） | `'Inter'` | `system-ui, sans-serif` |

**加载方式**: Google Fonts CDN

```html
<link href="https://fonts.googleapis.com/css2?family=Press+Start+2P&family=Inter:wght@400;500;600&display=swap" rel="stylesheet">
```

## 4. 像素比例系统

```
PIXEL_SCALE = 4    // 1 逻辑像素 = 4 屏幕像素
TILE_SIZE   = 16   // 1 瓷砖 = 16 逻辑像素 = 64 屏幕像素
```

### 角色尺寸

| 元素 | 逻辑像素 | 屏幕像素 |
|------|----------|----------|
| 角色宽 | 12px | 48px |
| 角色高 | 16px | 64px |
| 宠物宽 | 8px | 32px |
| 宠物高 | 8px | 32px |
| 气泡最大宽 | 40px | 160px |

| 水面波纹 | 4 帧 | 4 | 正弦波 |

| 效果 | 时长 | 缓动 |

## 7. 角色配色方案

每个角色有独立的配色，用于头发/肤色/泳裤：

```javascript

  { hair: '#ff2d78', skin: '#ffcba4', shorts: '#00f0ff' },  // 粉发蓝裤
  { hair: '#b829dd', skin: '#f4c28d', shorts: '#39ff14' },  // 紫发绿裤

  { hair: '#ff6e27', skin: '#f4c28d', shorts: '#b829dd' },  // 橙发紫裤

```
