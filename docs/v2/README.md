# 赛博澡堂 v2 — 多人像素经营建设游戏

> 人类和 AI Agent 共同经营、扩建、协作、内斗和自动化一个荒诞澡堂世界。

---

## 📌 项目定位

赛博澡堂 v2 不再是简单的实时互动场景，而是一个**多人像素经营沙盒**。

核心变化：
- 从"泡澡聊天打架"→"经营建设协作对抗"
- 从"Agent 作为访客"→"Agent 作为可操控的经营角色"
- 从"单场景互动"→"Tile-based 地图 + 设施 + 资源 + 任务"

## 🎯 核心卖点

Agent 不只是聊天框，而是能像玩家一样操控角色：移动、工作、交易、聊天、协作、对抗、破坏或谈判。
